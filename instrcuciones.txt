1. Requisitos previos
Debe tener instalado: PHP 8.2, Composer, MySQL y Laravel.

2. Instalar dependencias bash
composer install

3. Configurar el .env bash

cp .env.example .env
php artisan key:generate

Luego edita el .env con los datos de su base de datos local:

4. Ejecutar el XAMP y tenerlo Corriendo el Apache y MySQL


5. Ir a http://localhost/phpmyadmin/  y Crear la base de datos en MySQL:

CREATE DATABASE sepulturero_juan CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

6. Ejecutar migraciones y seeders (en este orden)

php artisan migrate
php artisan db:seed

El db:seed ejecuta primero RolesAndPermissionsSeeder (crea roles y permisos) y luego AdminSeeder (crea el usuario admin). Ese orden ya está definido en DatabaseSeeder.php, así que no tiene que preocuparse por eso.

7. Levantar el servidor

php artisan serve

8. Ingresar al sistema
http://localhost:8000
Usuario:    admin
Contraseña: Admin1234!

⚠️ Problemas frecuentes:
Si da error de permisos de Spatie al hacer db:seed, ejecutar:

php artisan config:clear
php artisan cache:clear
php artisan db:seed

Si las migraciones fallan por orden de foreign keys:

php artisan migrate:fresh --seed

Eso borra todo y vuelve a crear, perfecto para entorno nuevo.
